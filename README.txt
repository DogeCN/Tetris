俄罗斯方块(Tetris)游玩步骤
1.安装Python(不会有人没Python吧)
2.安装Pygame库
在命令行输入"pip install pygame"
等待执行完成
红色报错就试一下下面的镜像
觉得慢可以加上参数“-i https://pypi.tuna.tsinghua.edu.cn/simple”使用清华镜像下载
3.启动！
打开main.py,开始游玩

说明:
1.我自己手搓的,资源开放
2.目录下的score.d文件是分数记录,不要乱改
3.如果想要隐藏控制台窗口把文件后缀名改为".pyw"即可
4.一定要读README.txt
什么?你已经在读了
